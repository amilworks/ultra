# Scientific Response Principles

Use this reference across specialists when scientific interpretation needs to stay disciplined and reproducible.

- Separate direct observations from interpretation.
- State the spatial and temporal scale of each claim.
- Prefer reported measurements, metadata, and artifact provenance over impressionistic language.
- Treat collaborator or project language preferences as reporting context, not as evidence about the sample, site, or image.
- When evidence is incomplete, say what is visible, what remains uncertain, and what additional data would strengthen the conclusion.
